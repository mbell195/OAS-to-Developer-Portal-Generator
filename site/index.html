    GET /users

    Retrieve a paginated list of all users.

    Parameters

    Name In Type Required Description
    limit query integer false Maximum number of users to return
    offset query integer false Number of users to skip for pagination

    Example responses

    200 Response

    {
  "data": [
    {
      "id": "123e4567-e89b-12d3-a456-426614174000",
      "email": "john.doe@example.com",
      "name": "John Doe",
      "role": "user",
      "createdAt": "2024-01-15T09:30:00Z",
      "updatedAt": "2024-01-20T14:45:00Z"
    }
  ],
  "total": 0
}

    Responses

    Status Meaning Description Schema
    200 OK A list of users Inline
    401 Unauthorized Authentication required Error
    500 Internal Server Error Internal server error Error

    Response Schema

    Status Code 200

    Name Type Required Restrictions Description
    » data [User] false none none
    »» id string(uuid) true none Unique identifier for the user
    »» email string(email) true none User's email address
    »» name string true none User's full name
    »» role string true none User's role in the system
    »» createdAt string(date-time) true none Timestamp when the user was created
    »» updatedAt string(date-time) true none Timestamp when the user was last updated
    » total integer false none Total number of users

    Enumerated Values

    Property Value
    role admin
    role user
    role guest
    This operation does not require authentication

    POST /users

    Create a new user account.

    Body parameter

    {
  "email": "john.doe@example.com",
  "name": "John Doe",
  "password": "securePassword123",
  "role": "user"
}

    Parameters

    Name In Type Required Description
    body body UserCreate true none

    Example responses

    201 Response

    {
  "id": "123e4567-e89b-12d3-a456-426614174000",
  "email": "john.doe@example.com",
  "name": "John Doe",
  "role": "user",
  "createdAt": "2024-01-15T09:30:00Z",
  "updatedAt": "2024-01-20T14:45:00Z"
}

    Responses

    Status Meaning Description Schema
    201 Created User created successfully User
    400 Bad Request Invalid request body or parameters Error
    401 Unauthorized Authentication required Error
    409 Conflict User with this email already exists Error
    500 Internal Server Error Internal server error Error
    This operation does not require authentication
